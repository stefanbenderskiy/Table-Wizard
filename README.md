# Проект Table-Wizard (Яндекс лицей 2 год)
## Описание
Проект представляет из себя оконное приложение, которое имеет особый редактор таблиц, предназначенный для добавления или удаления данных таблицы, а также набор инструментов и функций, нужных для анализа и работы с данными из таблицы.
### > Основная суть
Приложение предназначено для удобной работы с таблицами.
### > Функционал
- Импортирование и экспортирование таблицы.
- Редактирование таблиц.
- Работа с данными таблицы.
- Анализ данных таблицы.
- Графическая визуализация данных таблицы.
## Технические задачи
1) Создать интерфейс приложения, импользуя библиотекуу PyQt
2) Создать систему взаимодействия с файловой системой (импортирования или экспортирования таблиц)
3) Создать редактор таблицы в приложении.
4) Добавить инструменты для работы с данными таблицы.
5) Добавить функционал, позволяющий анализировать данные таблицы
6) Графическая визуализация данных таблицы.
